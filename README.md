# Cabinet Augmenté

**Automatisation & IA pour les cabinets d'expertise comptable et d'audit.**

Je publie ici des outils open source conçus pour les professionnels du chiffre :
serveurs MCP, connecteurs métier, workflows IA — pour connecter vos outils
(logiciels de production, APIs, GED) directement à Claude, ChatGPT ou Mistral.

## Ce que vous trouverez ici

- 🔌 **mcp-cabinet** — Serveur MCP prêt à déployer pour connecter vos outils métier à vos chatbots IA
- 🧩 Connecteurs pour les APIs françaises (api.gouv.fr, Pappers, Gamma…)
- 📚 Ressources et guides pour les experts-comptables qui veulent aller plus loin
